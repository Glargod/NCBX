## 📋 Abstract

This repository contains the full honorary doctorate-level thesis and supporting simulations for a novel, **nature-prioritizing** treatment framework for schizophrenia.

We model schizophrenia as a **"leaky thalamus"** — impaired sensory gating in the thalamic reticular nucleus (TRN) caused by weak GABA braking and metabolic glucose dysregulation.

Using dynamical systems mathematics, we ran **10,000-patient Monte Carlo trials** comparing:
- Standard dopamine-based care
- Hybrid Smart Patch (LIFU + Keto 2.0)
- **Natural CBX-Enhanced Hybrid** ← **Winning Arm**

**Key Result**: The nature-first approach (closed-loop LIFU + ketogenic therapy + high-purity broad-spectrum hemp extract) achieved **85.5% mean symptom reduction** and **99.1% strong responders** in simulation.
